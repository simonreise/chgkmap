# Карта турниров ЧГК
Это - проект сайта с картой всех предстоящих очных ЧГК-турниров, написанный на node.js
Сайт состоит из двух частей:
* Фронтенда на Angular + Leaflet, находящегося в папке chgkmap
Фронтенд запускается командой `ng serve --open`
* Бэкенда на Express, находящегося в папке chgkmap-api
Бэкенд запускается командой `npm start`
Данные о турнирах загружаются с сайта рейтинга через API
